A brief history of Python

  • Python was conceptualized first in the late 80's by Guido Van Rossum. As time went on the language gradually added more features.
  • Python version 0.9.0 was released which had exception handling, classes, lists, and strings.
  • Python 2.0 was released in 2000 as a more open source project from the National Research Institute of Mathematics and Computer Science. This version had a full garbage collector and supported Unicode.
  • The newest version Python 3 was relased in 2008 with its latest version being released on July 8th 2019. Notably this version of Python is not compatible with older Python 2 code which has caused the lead developer Guido Van Rossum to extend support for the language until 2020.

Python as a programming language

  • Pythons largest draw as a language stems from its easy readability and overall gentle learning curve compared to other more cryptic-seeming languages such as c++ and java.
  • Python strives for easier readabilty with its tab-based system of defining blocks of code as opposed to some form of bracket.
  • Due to it being a high-level language it can be written much similar to human sounding language. An example of this being statements such as print(), reverse(), input() all doing exactly what their member names imply.
  • Instead of relying on knowing the syntax and tricks to a language to understand users can have an easier time understanding how to get the compiler to do what they want.
  • Python is a language that has always been designed to support object-oriented programming
