What is a Raspberry Pi?

  • A Raspberry Pi is a brand of single-board computers that are capable of most anything a regular computer can do and can be used as either an education tool to teach users languages like scratch or python.
  • Raspberry Pi was launched in 2012 and featured a single core 700 MHz CPU an 256 MB
  • Today, Raspberry Pi's have much better hardware with the Raspberry Pi 3B+ has a quad-core 1.4 MHz CPU and 1 GB of ram
  • Raspberry Pi's run off a linux distro called raspbian

What can a Raspberry Pi do?

  • The Raspberry Pi opens itself up to many different projects people can do either with a tutorial or just exploring the systems capabilities
  • First being the PiHole which is a DNS level ad blocker that prevents ads of any kind from appearing on any devices connected on your internet connection.
  • Next is the magic mirror which is a computer screen layered behind a mirror that can provide things like the weather, time, or even the daily news.
  • You can even use it as a host for a Web server and teach yourself HTML and CSS skills.
  • There are tons of different Projects that are available to anybody.
  • The Raspberry Pi is a very versitale machine that allows the user to test out their creativity or automate things on the cheap
